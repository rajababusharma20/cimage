Bachelor of Computer Application (BCA)

Duration : 3 Years

The program had been designed for students who have just passed 10+2 and are looking forward to a challenging career in the corporate world. The program aims at providing exposure to both IT and Management. After completion, candidate may either pursue higher studies or to for industry and business. India faces a huge scarcity of software professionals as per the recent NASSCOM survey. In fact, by 2015 the demand and supply gap is going to be as wide as 20 lac job.

Eligibility : 10+2 or Equivalent


First Year

Paper - I

Full Marks - 100 (Theory - 75, Practical - 25)

  • Computer Fundamental
  • Computer Software
  • Processing Modes
  • Programming Concept
  • Introduction to Computer Architecture
  • Practical

Paper - II

Full Marks - 100 (Theory - 75, Practical - 25)

  • Operating Systems
  • Foxpro : Viewing and Editing Data, Indexing, Sorting and Printing Report
  • Programming and Error Debugging
  • Multiple Data File and Custom Screens
  • Practical

Second Year

Paper - III

Full Marks - 100 (Theory - 75, Practical - 25)

  • Computer Networking
  • Internet
  • Boolean Algebra
  • Structured Programming
  • C-Language

Paper - IV

Full Marks - 100 (Theory - 75, Practical - 25)

  • Data Structure
  • Stack and Queues
  • Linked List; Tree; Searching and Sorting
  • Tables and graphs/li>
  • Computer Aided Drafting
  • Practical

Final Year

Paper - V

Full Marks - 100

  • Programming in Visual Basic
  • Mapping Project
  • Elements of the user Interface
  • Introduction to visual basic language
  • Working with forms
  • ActiveX Commands
  • The text box controls
  • List box and combo box; Dialog Box
  • Component of Visual Basic
  • Graphics with Visual Basic
  • Database Mng. System (DBMS)

Paper - VI

Full Marks - 100

  • Object Oriented Programming
  • Java Programming Overview of Java
  • Advance Topics in Computers

Paper - VII

Full Marks - 100

  • Design of a database for a business application
  • Design of data entry forms and report layouts for these databases
  • Creation of program to access and manipulate the databases
  • Development of a business application in RDBMS
  • Creating, merging, and deleting tables
  • Project preparation and vice versa

Paper - VIII

Full Marks - 100

  • Prepare a project in C++ programming
  • Project preparation and vice versa
